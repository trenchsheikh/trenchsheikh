**[Cloak](https://github.com/trenchsheikh/cloak)** &nbsp;·&nbsp; Privacy trading infrastructure on Solana

`Next.js` `Solana` `Rust` `Jupiter` `TypeScript`

**[AsterDex Trading Bot](https://github.com/trenchsheikh/asterdex-trading-bot)** &nbsp;·&nbsp; Telegram-native perps trading bot

Multi-user perps trading bot for Aster Finance. Handles wallet provisioning, dynamic leverage detection across 100+ symbols, real-time position management, and account ops — all via Telegram buttons. Projects built with this infrastructure have hit $1M+ trading volume.

`Node.js` `Telegram Bot API` `BEP-20` `Perps` `JavaScript`
